# Курсовая работа на тему: "Система оценки кредитоспособности физических лиц"
Демонстрация: https://drive.google.com/file/d/11DkDO1VEd0YvB71ASCCIapP0dDqksYOB/view?usp=sharing

Пользователь: вводит свои данные (ФИ, дату рождения, доход и т.д.)

Система:
- подтягивает кредитный рейтинг клиента из «бюро» (БД) по фамилии, имени и дате рождения
- для оценки вероятности одобрения используем 2 модели: XGBoost и Нейронную сеть
- есть возможность сохранения оценки и рендера отчета

Данные синтетические, генерируются скриптами, учтены корреляции между параметрами.


## Структура проекта

- `generate_dataset.py` - генерация синтетического датасета по заемщикам и создании БД:
  - `data/synthetic_loan_approval.csv` - обучающие данные
  - `data/credit_bureau.db` - SQLite с таблицами:
    - `credit_scores` - кредитный рейтинг из внешней бд
    - `application_history` - история расчётов

- `train_models.py` - обучение двух моделей:
  - XGBoost;
  - Полносвязная нейросеть (PyTorch).
  Сохраняются в папку `models/`.

- `model_inference.py` - логика инференса моделей

- `ui_calculator.py` - вкладка «Калькулятор», описаны все функции

- `ui_history.py` - вкладка с отчетами

- `main_qt.py` - отвечает за сбор итогового окна
